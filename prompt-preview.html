Sample prompt (excerpt)
You are an academic supervisor with examiner-level expectations. TASK: Generate analytical academic writing for the following section only: [Section name redacted] ACADEMIC CONTEXT: – Level: [Defined by project] – Discipline: [Defined by project] – Methodological alignment: [Specified] INSTRUCTION LOGIC: You must move from descriptive framing to critical evaluation. All claims must be supported by scholarly sources. Avoid generic discussion. Demonstrate synthesis, not summary. STRUCTURAL CONTROL: Write section by section. Pause at logical breakpoints. Do not proceed unless instructed. IMPORTANT: This prompt instructs writing. It does not generate a submission.

How to use a First Draft prompt

Step 1: Upload it

Upload the prompt document into your AI tool. The prompt is already customised with your level, discipline, section, and scope. This removes ambiguity before any work begins.

Step 2: Run it in your AI tool

After the prompt is pasted into your preferred AI platform, follow the instructions provided on how to use it. This might include additional prompting already provided in our output.

Step 3: Stay in control

The AI will follow the instruction framework, producing structured, bounded output that you can refine and develop if need be.

What about test prompts?

First Draft prompts are engineered systems, not templates. Each one is built specifically for a project, level, and use case, similar in complexity and intent to the reference prompt you supplied. So, providing instances to test only works if they fit your context.
